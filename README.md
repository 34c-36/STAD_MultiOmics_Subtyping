# 胃腺癌（STAD）多组学整合分子分型研究

## 项目简介
这是一个基于TCGA公共数据库的胃腺癌多组学整合分子分型项目，为个人学习完成。

## 项目背景
胃腺癌具有极强的瘤内异质性，传统TNM分期难以满足精准诊疗需求。本项目通过整合6维度多组学数据，构建更稳健的分子分型体系，实现胃癌患者的精准预后分层。

## 数据来源
所有数据均来自 [TCGA Xena公共数据库](https://xenabrowser.net/)，包括：
- 体细胞突变、拷贝数变异（CNV）、DNA甲基化
- mRNA表达、miRNA表达、蛋白表达
- 临床表型与生存数据

## 分析流程
1. **数据预处理**：完成6组学数据的质控、过滤、缺失值填充与标准化
2. **多组学整合**：采用相似性网络融合（SNF）算法整合多组学数据
3. **聚类分型**：基于谱聚类构建分子分型，通过轮廓系数确定最优分型数（K=8）
4. **分型验证**：
   - KM生存分析验证亚型预后差异（Log-rank p=0.035）
   - 特征基因热图可视化亚型分子特征
   - 与TCGA经典CIN/MSI/GS分型关联验证

## 技术栈
- R语言
- SNFtool（多组学整合）
- survival/survminer（生存分析）
- pheatmap（数据可视化）
- 无监督聚类、谱聚类

## 项目结果
### 1. 最优聚类数选择
基于轮廓系数确定最优分型数为8。

### 2. 生存预后分析
8个亚型的总生存期存在显著差异（p=0.035），分型可有效区分患者预后风险。

### 3. 分子特征热图
各亚型具有独特的基因表达模式，聚类边界清晰。

### 4. 与TCGA经典分型关联
本分型与TCGA经典CIN/MSI/GS分型高度一致，同时实现了对经典分型的细化。
